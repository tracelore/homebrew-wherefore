# Tracelore Wherefore

## How do I install these formulae?

`brew install tracelore/wherefore/<formula>`

Or `brew tap tracelore/wherefore` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tracelore/wherefore"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
